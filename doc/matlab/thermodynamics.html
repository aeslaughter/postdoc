Thermodynamic Relationships Test Function

Section 4 of Zabaras and Samanta (2004) details the thermodynamic relationships that relate temperature (\(\theta\)), volume fraction (\(\epsilon\)), and liquid concentration (\(C_l\)). This function is designed to test that these relationships are working properly. All equation references refer to Samanta and Zabaras (2005).

function thermodynamics

% THERMODYNAMICS
% Test for thermodynamic relationships of Samanta and Zabaras 2005
%
% Syntax:
%   thermodynamics;

Gather the constants and element data

C = liquid concentration (non-dimensional) T = nodal temperatuer (non-dimensional)

[ ~, T] = parameters;
p = quad4;

Compute and dislpay the nodal values

for i = 1:length(C);

    % Display the current point
    str = sprintf('ELEMENT 0; Point %i (%g, %g)', i, p(i,1), p(i,2)); disp(str);

    % Display the thermodynamic values at this point
    out('theta_m', theta_m(const));
    out('theta_liquidus', theta_liquidus(C(i),const));
    out('theta_solidus', theta_solidus(C(i),const));
    out('h_liquidus', h_liquidus(C(i),const));
    out('h_solidus', h_solidus(C(i),const));
    out('h_e', h_e(C(i),const));
    out('epsilon_e', epsilon_e(C(i),const));

    % Compute the initial enthalpy
    e = epsilon_initial(T(i),C(i), const);
